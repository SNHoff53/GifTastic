# GifTastic

## Overview
This is app is a fun way to keep your favorite gifs! You can also choose from pre-populated gifs or add your own to the queue using the search bar. A limit of 10 random gifs will appear below.

## Technologies Used: 
- HTML
- CSS
- JavaScript
- jQuery
- AJAX
- Giphy API
